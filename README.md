# TA-dnslookup

## Enrich your Splunk searches with DNS query results for any record type from any DNS server.   


### Lookups:  
- dnslookup
- dnslookup_mx
- dnslookup_reverse
- dnslookup_ptr (Same as _reverse but here for conventions)
- dnslookup_ns
- dnslookup_aaaa
- dnslookup_txt
- dnslookup_cname
- dnslookup_alias
- dnslookup_soa

See transforms.conf for details.
